# Node API Demo

Demo REST API built with Node.js. 

## Tech Stack
* Node.js 8.x.x
* NPM Libraries
    - Nodemon for Debugging and keeping track of changed files
    - Express for initiating the server
    - Mongoose for interracting with MongoDB
    - BodyParser to parse request body
* MongoDB 4.0
